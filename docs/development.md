# Development

Developer documentation will be rebuilt alongside the CNC refactor.

For now, the important runtime names are:

- Config file: `klipper_screen.conf`
- User config directory: `~/.config/klipper_screen`
- Virtual environment: `~/.klipper-screen-env`
- Systemd service: `klipper-screen.service`
- Expected source directory: `~/klipper-screen-cnc`

Install dependencies with:

```sh
pip install -r touch-ui/scripts/KlipperScreen-requirements.txt
```
